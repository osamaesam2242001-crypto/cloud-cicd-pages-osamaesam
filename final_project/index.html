What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a modern development practice that helps automate testing and deployment. Continuous Integration ensures that code changes are collected and checked regularly. Continuous Deployment publishes the latest version of the website automatically. In this project, GitHub Actions handles the automation, while GitHub Pages hosts the final website.

My Pipeline

When I push my project files to the main branch on GitHub, the workflow starts automatically. GitHub Actions prepares the site files and deploys them to GitHub Pages. After the process finishes successfully, the updated website becomes available online.
